# SiteNav

一个基于 Cloudflare Workers 的简易网站导航工具。

## 功能特点

- **安全管理后台**：密码保护，防止未授权访问
- **响应式设计**：完美适配桌面和移动设备
- **轻量高效**：基于 Cloudflare Workers 边缘计算
- **安全可靠**：SHA-256 密码加密 + 代码混淆保护

## 技术架构

- **运行时**：Cloudflare Workers
- **存储**：Workers KV
- **安全**：SHA-256 密码加密 + 代码混淆
- **前端**：HTML5 + CSS3 + JavaScript
